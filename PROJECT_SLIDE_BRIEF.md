# สรุปข้อมูลโครงการสำหรับทำสไลด์: หรีดร่วมบุญ

## 1. ชื่อโครงการ

**หรีดร่วมบุญ (Hreed Ruam Bun)**  
แพลตฟอร์มหรีดดิจิทัลเพื่อร่วมอาลัย ร่วมทำบุญ และลดขยะจากพวงหรีด

## 2. แนวคิดหลัก

หรีดร่วมบุญเป็นแพลตฟอร์มสำหรับงานศพไทยพุทธ ที่เปลี่ยนการมอบพวงหรีดจากสิ่งของที่ใช้ครั้งเดียวแล้วกลายเป็นขยะ ให้เป็นการบริจาคเงินร่วมบุญผ่านระบบดิจิทัล พร้อมแสดงรายชื่อผู้ร่วมบุญบนบอร์ดหน้างานและส่งการ์ดขอบคุณกลับให้ผู้บริจาค

โครงการนี้ช่วยให้เจ้าภาพจัดการรายชื่อผู้ร่วมบุญได้ง่าย ศูนย์หรือผู้ดูแลงานตรวจสอบสลิปและพิมพ์ป้ายได้เป็นระบบ และผู้บริจาคสามารถร่วมแสดงความอาลัยได้สะดวกผ่านมือถือ

## 3. ปัญหาที่ต้องการแก้

- พวงหรีดจริงจำนวนมากกลายเป็นขยะหลังจบพิธี
- เจ้าภาพต้องจัดการรายชื่อผู้ส่งหรีดและยอดบริจาคด้วยมือ
- การตรวจสลิป การบันทึกยอด และการพิมพ์ป้ายหน้างานมีโอกาสผิดพลาด
- ผู้ร่วมงานบางคนไม่สะดวกเดินทาง แต่ยังต้องการแสดงความอาลัย
- ศูนย์หรือผู้ดูแลงานต้องการระบบที่ติดตามสถานะงานได้ชัดเจน

## 4. วิธีแก้ปัญหา

แพลตฟอร์มสร้างหน้ารำลึกเฉพาะงานศพแต่ละงาน มี QR สำหรับรับเงินบริจาค ระบบแนบสลิป ระบบตรวจสอบโดยเจ้าหน้าที่ ระบบพิมพ์ป้ายชื่อผู้ร่วมบุญ และหน้าการ์ดขอบคุณหลังบริจาค

การทำงานทั้งหมดออกแบบให้เป็น mobile-first สำหรับผู้ใช้งานชาวไทย โดยใช้โทนสีครีม ทอง และเขียวมรกต เพื่อให้เหมาะกับบริบทงานอาลัยและยังใช้งานง่ายสำหรับผู้สูงอายุ

## 5. กลุ่มผู้ใช้งานหลัก

**ผู้บริจาค**  
เปิดหน้ารำลึก เลือกบริจาค โอนเงิน แนบสลิป กรอกชื่อสำหรับป้าย และรับ e-card หรือใบยืนยันการร่วมบุญ

**เจ้าภาพ**  
ดูสรุปงาน รายชื่อผู้ร่วมบุญ เอกสารเจ้าภาพ และข้อมูลปิดงาน

**ศูนย์หรือผู้ดูแลงาน**  
สร้างงาน เปิด/ปิดงาน ตรวจสลิป พิมพ์ป้ายชื่อ ติดตามรายการค้าง และจัดการงานประจำวันของศูนย์

**ผู้ดูแลระบบกลาง**  
ดูภาพรวมข้ามศูนย์ รายงาน ESG รายงานการเงิน สถานะระบบ ผู้ใช้ ศูนย์ และ audit log

## 6. Flow ผู้บริจาค

1. เข้าหน้า `/{slug}` เพื่อดูข้อมูลผู้วายชนม์และกำหนดการ
2. ไปหน้า `/{slug}/payment` เพื่อดู QR/เลขบัญชี โอนเงิน และแนบสลิป
3. ไปหน้า `/{slug}/verifying` หลังแนบสลิป เพื่อเข้าสู่สถานะตรวจสอบ
4. ไปหน้า `/{slug}/print-name` เพื่อกรอกชื่อหรือองค์กรสำหรับป้าย
5. ไปหน้า `/{slug}/ecard` เพื่อรับการ์ดขอบคุณ ใบยืนยัน และภาพจำลองการมอบหรีด

## 7. ฟีเจอร์สำคัญ

- หน้ารำลึกเฉพาะแต่ละงานศพ
- QR PromptPay และข้อมูลบัญชีสำหรับรับบริจาค
- อัปโหลดสลิปโอนเงิน
- ระบบตรวจสอบสลิปโดยคน ไม่ยืนยันเงินอัตโนมัติจาก AI
- ระบบกรอกชื่อและพิมพ์ป้ายหน้างาน
- e-card ขอบคุณและใบยืนยันการร่วมบุญ
- AI photo flow สำหรับภาพจำลองการมอบหรีด
- Dashboard ศูนย์สำหรับงานประจำวัน
- Dashboard เจ้าภาพสำหรับดูสรุปและรายชื่อ
- Dashboard Admin สำหรับภาพรวม รายงาน ESG ผู้ใช้ ศูนย์ และ audit

## 8. จุดเด่นของโครงการ

- ลดขยะจากพวงหรีดจริงและสนับสนุนแนวคิด zero waste
- เหมาะกับบริบทงานศพไทยพุทธ ทั้งภาษา ภาพลักษณ์ และขั้นตอนใช้งาน
- แยกบทบาทชัดเจนระหว่างผู้บริจาค เจ้าภาพ ศูนย์ และแอดมินกลาง
- ออกแบบให้ใช้งานง่ายบนมือถือเป็นหลัก
- รองรับงานปฏิบัติจริง เช่น ตรวจสลิป พิมพ์ป้าย ปิดงาน และทำรายงาน
- ให้ความสำคัญกับความปลอดภัยของข้อมูลและการยืนยันธุรกรรมโดยมนุษย์

## 9. เทคโนโลยีที่ใช้

- **Next.js App Router** สำหรับเว็บแอปและ route หลัก
- **TypeScript** เพื่อความแม่นยำของข้อมูลและโค้ด
- **Supabase** สำหรับฐานข้อมูล Storage และ server-side admin access
- **Tailwind CSS v4** สำหรับ UI mobile-first
- **PromptPay QR** สำหรับการชำระเงิน
- **lucide-react** สำหรับ icon ในระบบ
- **AI photo flow** สำหรับประสบการณ์เสริมหลังการร่วมบุญ

## 10. ความปลอดภัยและข้อกำกับสำคัญ

- ไม่ยืนยันยอดบริจาคจาก AI หรือการอ่านภาพเพียงอย่างเดียว
- การยืนยันเงินต้องมีมนุษย์ตรวจสอบ
- Service role ของ Supabase ใช้เฉพาะ server-side
- ไม่เปิดเผยข้อมูลข้ามศูนย์ในหน้าที่เป็น center-scoped
- Admin กลางเน้นดูภาพรวมและกำกับดูแล ไม่ทำหน้าที่เป็นคิวปฏิบัติการรายงานศพแทนศูนย์
- หลีกเลี่ยงการลบข้อมูลหรือ action ทำลายข้อมูล หากไม่ได้รับคำสั่งชัดเจน

## 11. โครงสไลด์แนะนำ

### Slide 1: Title
หรีดร่วมบุญ  
แพลตฟอร์มหรีดดิจิทัล ร่วมอาลัย ร่วมทำบุญ ลดขยะ

### Slide 2: Problem
พวงหรีดจริงสร้างขยะจำนวนมาก และการจัดการรายชื่อ/สลิปในงานศพยังใช้แรงงานมือสูง

### Slide 3: Solution
เปลี่ยนการส่งหรีดเป็นการบริจาคผ่าน QR พร้อมระบบแนบสลิป พิมพ์ป้าย และการ์ดขอบคุณ

### Slide 4: User Journey
ดูหน้างาน → โอนเงิน → แนบสลิป → กรอกชื่อป้าย → รับ e-card

### Slide 5: Roles
ผู้บริจาค / เจ้าภาพ / ศูนย์ / Admin กลาง พร้อมหน้าที่แยกชัดเจน

### Slide 6: Key Features
PromptPay QR, slip upload, nameplate printing, e-card, certificate, AI photo, dashboards

### Slide 7: Operations
ศูนย์ตรวจสลิป พิมพ์ป้าย ดูรายการค้าง เปิด/ปิดงาน และติดตามงานประจำวัน

### Slide 8: ESG Impact
ลดขยะจากพวงหรีดจริง เปลี่ยนเป็นเงินบริจาคและข้อมูลวัดผลได้

### Slide 9: Tech Stack
Next.js, TypeScript, Supabase, Tailwind CSS, PromptPay QR, AI photo flow

### Slide 10: Vision
สร้างมาตรฐานใหม่ของการร่วมอาลัยที่สุภาพ เรียบง่าย ตรวจสอบได้ และเป็นมิตรต่อสิ่งแวดล้อม

## 12. ข้อความสั้นสำหรับ Pitch

หรีดร่วมบุญคือแพลตฟอร์มหรีดดิจิทัลสำหรับงานศพไทย ที่ช่วยให้ผู้ร่วมงานบริจาคแทนการส่งพวงหรีดจริงผ่าน QR แนบสลิป กรอกชื่อสำหรับป้าย และรับการ์ดขอบคุณได้ครบในมือถือเดียว ระบบช่วยเจ้าภาพและศูนย์จัดการงานหลังบ้านอย่างเป็นระเบียบ พร้อมลดขยะและสร้างข้อมูล ESG ที่ตรวจสอบได้
