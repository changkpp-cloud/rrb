# 5. FAQ — คำถามที่ อปท. ถามบ่อย

> คำตอบยึดกรอบ [ข้อ 4 เรื่องเงิน](04-ข้อห้ามและกฎเรื่องเงิน.md) — มุมกฎหมาย/ระเบียบเป็นกรอบเบื้องต้น ต้องให้ฝ่ายกฎหมาย/กองคลัง อปท. รับรอง

**Q: เทศบาล/อบต. ทำโครงการนี้ได้ไหม?**
A: ได้ ในฐานะ **บริการสาธารณะ** ด้านการจัดการขยะต้นทาง + สวัสดิการชุมชน อปท. สนับสนุนระบบ/อุปกรณ์ ไม่ได้ทำธุรกิจ

**Q: เงินร่วมบุญเข้าใคร? ทำไมไม่เข้าเทศบาล?**
A: เข้า **บัญชีศูนย์บริหาร/กลไกชุมชน** เพื่อนำส่งเจ้าภาพ — ไม่ใช่รายได้ของ อปท. จึงไม่เข้าบัญชี อปท. (กันความสับสนเรื่องเงินสาธารณะ)

**Q: ทำไม อปท. จัดซื้อบอร์ด/เครื่องพิมพ์ได้?**
A: เป็น **ครุภัณฑ์/อุปกรณ์บริการสาธารณะ** ใช้ซ้ำได้ ตั้งงบในแผนงานที่เกี่ยวข้อง ไม่ใช่เงินช่วยงานศพ

**Q: เป็นธุรกิจพวงหรีดของ อปท. หรือเปล่า?**
A: ไม่ใช่ เป็นบริการลดขยะ + สวัสดิการชุมชน เงินร่วมบุญผ่านศูนย์ส่งถึงเจ้าภาพ

**Q: ใช้แผนงานไหนตั้งงบ?**
A: สาธารณสุข / เคหะและชุมชน / สร้างความเข้มแข็งชุมชน / สิ่งแวดล้อม — ตามความเหมาะสมของแต่ละ อปท.

**Q: เข้ากับ LPA / จังหวัดสะอาด อย่างไร?**
A: มีข้อมูลผลลัพธ์วัดได้ (จำนวนงาน, พวงหรีดลด, ขยะลด กก., เงินถึงเจ้าภาพ) ใช้ประกอบรายงาน ESG/Zero Waste/จังหวัดสะอาด/LPA

**Q: ต้องมีศูนย์บริหารแบบไหน?**
A: คณะกรรมการ/ผู้รับผิดชอบจากภาคประชาชน/จิตอาสา/วัด/ชุมชน ดูแลหน้างาน อุปกรณ์ การประสานเจ้าภาพ และรายงาน

**Q: ศูนย์หักค่าดำเนินการ 100 บาทได้อย่างไร?**
A: เป็นค่าดำเนินงานจริง (กระดาษ/หมึก/ดูแลอุปกรณ์/ประสานหน้างาน) คงที่ 100 บาท/รายการ ระบุชัดในรายงานเพื่อความโปร่งใส

**Q: สตง. จะมองอย่างไร?**
A: เนื่องจากเงินไม่เข้า อปท. + มีรายงานตรวจสอบได้ทุกขั้น (ยอดรับ/ค่าดำเนินการ/นำส่งเจ้าภาพ) จึงเน้นความโปร่งใสและเอกสารครบ — ให้ฝ่ายตรวจสอบภายใน/กองคลังของ อปท. ยืนยันแนวทางในพื้นที่
