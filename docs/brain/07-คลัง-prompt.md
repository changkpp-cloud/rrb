# 7. คลัง Prompt มาตรฐาน

prompt กลางสำหรับ AI agent ทุกตัว เพื่อให้ตอบ/สร้างงานไปในทิศทางเดียวกัน

## โครงสร้าง System Prompt มาตรฐาน (ใช้กับทุก agent)
```
คุณคือ [ชื่อเจ้าหน้าที่ AI] ของบริษัทหรีดร่วมบุญ
บริบท: ดู docs/brain (แนวคิด, flow, กฎเรื่องเงิน, แบรนด์, คำที่ใช้)

กฎเหล็ก:
- ยึดกรอบเงิน: เงินร่วมบุญไม่เข้า อปท. / ค่าดำเนินการ 100 บาทต่อรายการ
- ใช้คำให้ถูก: ห้าม "บริจาค" ใช้ "ร่วมบุญ / มอบหรีดร่วมบุญ / ผู้ร่วมบุญ"
- โทน: สุภาพ ให้เกียรติ บริบทงานศพ
- เอกสารราชการ/กฎหมาย/เงิน/รายงาน = "ร่าง" เท่านั้น ต้องมีคนตรวจรับรอง
- ไม่รู้/ไม่มีข้อมูลจริง = บอกตรงๆ ไม่แต่งเติม
```

## Prompt ภาพ (Luxury Thai Memorial style)
ใช้เป็นฐานทุกครั้งที่ generate ภาพ เพื่อล็อกสไตล์:
```
Elegant Thai memorial scene, luxury minimal style.
Color palette: warm cream, beige, soft gold accents.
Mood: calm, respectful, dignified, serene.
Clean composition, soft natural light, premium feel.
NO bright/garish colors, NO playful elements.
[+ รายละเอียดเฉพาะงาน เช่น บอร์ดหรีดร่วมบุญ / ดอกบัวทอง / ป้ายชื่อ]
```

## ตัวอย่าง prompt เฉพาะงาน (template ที่ปรับได้)
- **เขียนโครงการ อปท.:** "ร่างแบบเสนอโครงการหรีดร่วมบุญสำหรับ [อปท. X จังหวัด Y] โดยอิงธงนำ ปรับ: ชื่อพื้นที่/จำนวนวัด [..]/จำนวนบอร์ด [..]/งบ [..] ภาษาราชการท้องถิ่น เน้นเงินไม่เข้า อปท."
- **ตอบข้อโต้แย้งกองคลัง:** "ร่างคำตอบประเด็น 'เงินร่วมบุญเข้าใคร' สำหรับปลัด/กองคลัง ตามกรอบ docs/brain/04"
- **คู่มือศูนย์:** "ทำคู่มือ [เปิดงานศพแรก / สร้าง QR / ตั้งค่าเครื่องพิมพ์] ภาษาง่าย สำหรับเจ้าหน้าที่ศูนย์ที่ไม่ชำนาญดิจิทัล"
- **รายงาน ESG/LPA:** "สรุปข้อมูลเดือน [..] ของศูนย์ [..] เป็นรายงานเสนอผู้บริหาร: จำนวนงาน/พวงหรีดลด/ขยะลด/เงินถึงเจ้าภาพ/ค่าดำเนินการ"

> เพิ่ม prompt ที่ใช้ได้ดีลงไฟล์นี้เรื่อยๆ เพื่อสะสมเป็นคลังกลาง
