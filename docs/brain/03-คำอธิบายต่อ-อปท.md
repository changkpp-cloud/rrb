# 3. คำอธิบายต่อ อปท. — พูดกับใคร พูดมุมไหน

ปรับสารให้ตรงกับคนฟังแต่ละกลุ่ม (ใช้ทำ Pitch/Script/FAQ/สไลด์)

## นายก อปท. / ผู้บริหาร
มุม: **ผลงานจับต้องได้**
- ลดขยะในพื้นที่อย่างเป็นรูปธรรม + ช่วยประชาชน (เจ้าภาพ)
- สร้างภาพลักษณ์ผู้นำที่ใช้นวัตกรรมดูแลประชาชนถึงวาระสุดท้าย
- มี Dashboard/รายงานผลไว้แถลงผลงาน

## ปลัด / กองคลัง
มุม: **ความถูกต้องด้านการเงิน** (ด่านสำคัญสุด)
- **เงินร่วมบุญไม่เข้าบัญชี อปท.** — ผ่านบัญชีศูนย์บริหาร/กลไกชุมชน
- อปท. สนับสนุน **ครุภัณฑ์/อุปกรณ์/ระบบ** ไม่ใช่เงินหมุนเวียนหรือเงินช่วยงานศพ
- ตรวจสอบได้ โปร่งใส มีรายงาน (รองรับมุมมอง สตง.)

## กองสาธารณสุขและสิ่งแวดล้อม
มุม: **ลดขยะต้นทาง**
- ลดพวงหรีดสด/โฟม/พลาสติก ลดภาระวัดและชุมชนในการเก็บ/กำจัด
- ข้อมูลประกอบ Zero Waste / จังหวัดสะอาด

## กองสวัสดิการสังคม
มุม: **สวัสดิการ**
- สวัสดิการผู้วายชนม์/ชุมชน ลดภาระค่าใช้จ่ายเจ้าภาพ
- ชุมชนดูแลกันเองในวาระสุดท้าย

## สภาท้องถิ่น
มุม: **ประโยชน์ประชาชน + งบสมเหตุผล**
- งบไม่สูง (โครงการนำร่อง ≤ 500,000 บาท เป็นอุปกรณ์/ระบบ ไม่ใช่เงินหมุนเวียน)
- อุปกรณ์ใช้ซ้ำได้ มีรายงานผลที่วัดได้
