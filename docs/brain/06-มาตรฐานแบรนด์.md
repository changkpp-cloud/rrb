# 6. มาตรฐานแบรนด์

## ชื่อ
- ชื่อแบรนด์/โครงการ: **หรีดร่วมบุญ** (Zero Waste)
- ชื่อนิติบุคคล (ภาษาอังกฤษ): **READRUAMBOON** — มีเอกสารจองชื่อแล้ว ต้องยื่นจดทะเบียน **ภายใน 20/07/2569**
- โดเมน/เว็บ: `rrb.center`

## โลโก้
- โลโก้ **ดอกบัวทอง** (LotusIcon / ไฟล์ `rrb-logo-removebg-preview.png`)
- สื่อความ: ความสงบ ระลึกถึง บุญ สุภาพ สมเกียรติ

## สี (โทนแบรนด์)
- **ทอง (gold)** = primary brand (`gold-50 … gold-900`)
- **ครีม/เนย (cream)** = พื้นหลัง/การ์ด (`cream-50 … cream-200`)
- หลีกเลี่ยงสีฉูดฉาด/โทนสนุกในบริบทงานศพ (โทนต้องสงบ ให้เกียรติ)

## ฟอนต์
- **Sarabun** (ไทย+ละติน) ทุกส่วน รวม e-card

## โทนภาพ (Image / สื่อ)
**"Luxury Thai Memorial"** — ครีม เบจ ทอง สุภาพ สมเกียรติ สะอาดตา
- ใช้กับ: สไลด์ อินโฟกราฟิก โปสเตอร์ ภาพบอร์ด ภาพ UI
- ภาพ AI ใช้คลัง prompt มาตรฐาน (ดู [ข้อ 7](07-คลัง-prompt.md)) เพื่อไม่ให้หลุดสไตล์

## โทนภาษา (Voice)
- สุภาพ ให้เกียรติ อบอุ่น น่าเชื่อถือ
- ไม่ขายของโฉ่งฉ่าง ไม่ใช้คำเร่งเร้าแบบโฆษณาทั่วไป
- คำที่ใช้/ห้ามใช้ → ดู [ข้อ 8 อภิธานศัพท์](08-อภิธานศัพท์.md)

## บริบทอ่อนไหว
งานศพ/การสูญเสีย — สื่อสารด้วยความเคารพเสมอ หลีกเลี่ยงมุกตลก ภาพหวือหวา หรือคำที่ลดทอนความสมเกียรติ
