# 🧠 สมองกลาง (Knowledge Base) — บริษัทหรีดร่วมบุญ

แหล่งความรู้กลาง (Single Source of Truth) ของบริษัท สำหรับให้ **AI Agent ทุกตัว** และทีมงานดึงไปใช้ เพื่อให้ตอบ/สร้างเอกสารไปในทิศทางเดียวกัน ไม่หลุดคำ ไม่หลุด flow ไม่อธิบายผิด

## ใช้ยังไง
- ก่อน AI agent ตอบลูกค้า/สร้างเอกสาร/ทำสื่อ → ให้อ้างอิงไฟล์ในโฟลเดอร์นี้เป็นความจริงตั้งต้น
- ข้อเท็จจริงเปลี่ยน → **แก้ที่นี่ที่เดียว** (ไฟล์เหล่านี้ = สถานะปัจจุบัน เหมือน `CLAUDE.md`)
- ประวัติการเปลี่ยนแปลงดูที่ `CHANGELOG.md` ของ repo

## ความสัมพันธ์กับเอกสารอื่น
| ไฟล์ | เก็บอะไร |
|---|---|
| `docs/brain/*` (ที่นี่) | ความรู้ **ธุรกิจ + การสื่อสาร + แบรนด์** (ให้ AI agent ใช้) |
| `CLAUDE.md` | ความรู้ **เทคนิค/โค้ด** (สถานะระบบปัจจุบัน) |
| `CHANGELOG.md` | ประวัติการแก้ไขสะสม |
| แบบเสนอโครงการ (ธงนำ) | เอกสารต้นฉบับเสนอ อปท. |

## ⚠️ Governance (สำคัญ — ลูกค้าคือราชการ)
เนื้อหาฝั่ง **เอกสารราชการ / ระเบียบ-กฎหมาย / เงิน / รายงานทางการ** — AI ใช้ "ร่าง" ได้ แต่ต้องมี **คนตรวจและรับรอง** ก่อนส่งจริงเสมอ (โดยเฉพาะมุมที่เกี่ยวกับ สตง. และเงินไม่เข้า อปท.) ไฟล์กลุ่มกฎหมาย/ระเบียบเป็น "กรอบเบื้องต้น" ไม่ใช่ความเห็นทางกฎหมายขั้นสุดท้าย
