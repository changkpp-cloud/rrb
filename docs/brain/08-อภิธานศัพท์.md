# 8. อภิธานศัพท์ / คำที่ใช้

## คำที่ "ห้ามใช้" → ใช้แทน
| ❌ ห้ามใช้ | ✅ ใช้แทน |
|---|---|
| บริจาค / เงินบริจาค | **ร่วมบุญ / ร่วมบุญเป็นเงิน / มอบหรีดร่วมบุญ** |
| ผู้บริจาค | **ผู้ร่วมบุญ / ผู้มอบหรีดร่วมบุญ** |
| ขายของ/โปรโมชัน (โทนโฆษณา) | สื่อสารแบบ**ให้เกียรติ สุภาพ** |

เหตุผล: "บริจาค" สื่อความผิดบริบทและเลี่ยงประเด็นเงินสาธารณะ — โครงการนี้คือ "ร่วมบุญ" ผ่านศูนย์

## คำเรียกหลัก
- **ผู้วายชนม์** — ผู้เสียชีวิต (เจ้าของงาน)
- **เจ้าภาพ** — ครอบครัวผู้จัดงาน (ผู้รับเงินสุทธิ)
- **ผู้ร่วมบุญ** — แขกที่ร่วมบุญเป็นเงิน + ฝากชื่อขึ้นป้าย
- **ศูนย์บริหารหรีดร่วมบุญประจำ อปท.** — กลไกชุมชนที่บริหารหน้างาน (เรียกสั้น: "ศูนย์")
- **บอร์ดหรีดร่วมบุญ** — บอร์ดใช้ซ้ำสำหรับติดป้ายชื่อ แทนพวงหรีดสด
- **ป้ายชื่อ / ป้ายอาลัย** — ป้ายที่พิมพ์ชื่อผู้ร่วมบุญไปติดบนบอร์ด
- **อปท.** — องค์กรปกครองส่วนท้องถิ่น (เทศบาล/อบต.) = ลูกค้า/เจ้าของโครงการประจำพื้นที่

## ศัพท์ผลลัพธ์ (ESG)
- **พวงหรีดที่ลด** = จำนวนผู้ร่วมบุญ (1 ราย ≈ ลดพวงหรีดสด 1 พวง)
- **ขยะที่ลด** = พวงหรีดที่ลด × 2 กก.
- **ค่าดำเนินการ** = 10% ของยอดร่วมบุญรวม (รายได้ศูนย์)
- **ยอดนำส่งเจ้าภาพ (สุทธิ)** = ยอดร่วมบุญรวม × 90% (= ยอดรวม − ค่าดำเนินการ)
